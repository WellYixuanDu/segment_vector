# segment_vector
## 目录结构
- 源代码
  - `include/`: 包含源代码
    - `segment_vector.h`: segment_vector源代码文件
  - `test/`: 包含测试代码
    - `acc_test.cc`: 对`segment_vector`进行正确性测试的测试文件 
    - `speed_test.cc`: 进行容器间的性能对比

- 运行文件
  - `CMakeLists.txt`
  - `run.sh`: 运行脚本 
- 文档
  - `README.md`

## 项目运行
执行 `./run.sh` 脚本，即可自动编译运行
